# Solana Escrow Program

🚀 Build Secure Token Swaps on Solana | Rust Anchor Framework

Learn how to build a complete Solana escrow program for secure token swaps using the Anchor framework! This project demonstrates real-world blockchain development with practical smart contract implementation.

🚀 What I've Learn:

- Solana program development with Rust & Anchor
- Escrow logic for secure token exchanges
- PDA (Program Derived Address) management
- Token vault creation and management
- Testing and deployment on Solana Devnet

🛠 Tech Stack:
Solana • Rust • Anchor • TypeScript • Node.js • Solana Kit

🎓 Perfect for developers learning Solana smart contract development and understanding DeFi primitives on Solana!

## Developer Notes

**Primary Focus:** Rust Smart Contract Development
- All blockchain logic is implemented in Rust using the Anchor framework
- Smart contracts are located in the `programs/` directory
- Rust code handles all on-chain operations including escrow creation, token deposits, and swaps

**Development Workflow:**
1. Write and test Rust smart contracts in `programs/escrow/`
2. Generate TypeScript client using Codama
3. Test using Node.js and npm
4. Deploy using Anchor CLI

This project serves as an excellent foundation for understanding DeFi primitives on Solana while following modern development practices.

## Quick Start

1. Clone the repository:
   ```bash
   git clone https://github.com/mikemaccana/anchor-escrow-2025.git
   cd anchor-escrow-2025
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Deploy the program:
   ```bash
   anchor deploy
   ```
